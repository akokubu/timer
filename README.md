# Timer (ストップウォッチ/カウントダウン)

ブラウザだけで動く軽量なタイマーアプリです。ストップウォッチとカウントダウンを切り替えて利用できます。初期モードはカウントダウンです。

## 機能
- ストップウォッチ/カウントダウン切替
- 開始/一時停止/再開、リセット
- ストップウォッチ時のラップ記録
- カウントダウン終了時のビープ音と点滅
- 5分プリセットボタン（カウントダウン）

## 使い方
1. `index.html` をブラウザで開きます。
2. モードを切り替える場合はヘッダーのトグルを使用。
3. カウントダウンでは分・秒を入力、または「5分」ボタンを押して開始。

- キーボード操作: Space=開始/一時停止、R=リセット、L=ラップ（SW時）

## ファイル構成
- `index.html` … 画面レイアウト
- `style.css` … スタイル
- `script.js` … ロジック
- `timer.jpg` … ヘッダー/ファビコン用アイコン（`timer.png` から変換）

## 開発メモ
- 高精度計測: `performance.now()` を使用
- レンダリング: `requestAnimationFrame`
- 音声: Web Audio API（`AudioContext`）
